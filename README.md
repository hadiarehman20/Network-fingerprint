Network Fingerprint Project

📌 Description

This is a Flask-based web application that captures and analyzes network traffic to generate behavioral fingerprints of websites.

⚙️ Features

- Capture network data
- Extract features
- Classify traffic
- Generate fingerprints

🚀 How to Run

1. Install dependencies:
   pip install flask
2. Run the app:
   python app.py
3. Open browser:
   http://127.0.0.1:5000/

📁 Project Structure

- app.py → Main Flask app
- capture.py → Captures network data
- extract.py → Extracts features
- classify.py → Classification logic
- fingerprint.py → Generates fingerprint
- templates/ → HTML files
- static/ → CSS & JS

🛠️ Technologies Used

- Python
- Flask
- HTML/CSS/JavaScript
